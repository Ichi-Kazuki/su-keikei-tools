# 学びキーワード診断

オープンキャンパス向けの小さなWeb診断です。来場者はキーワードを3つまで選び、経済経営学部の学び領域、関連する問い、授業、相談で聞くとよいことを確認できます。

## ファイル

- `Code.gs`: Apps Scriptのサーバー側コード
- `Index.html`: 来場者が見るWeb画面
- `appsscript.json`: Apps Scriptの設定
- `preview.html`: ローカルで画面を確認するための静的プレビュー
- `data/learning_keyword_diagnosis_data.xlsx`: 管理用シートの雛形

## 使い方

1. Googleスプレッドシートを新規作成します。
2. `拡張機能 > Apps Script` を開きます。
3. `Code.gs` の内容を貼り付けます。
4. HTMLファイル `Index` を追加し、`Index.html` の内容を貼り付けます。
5. 必要なら `appsscript.json` を表示して、このフォルダの内容に合わせます。
6. Apps Scriptで `setupDiagnosisSheets` を一度実行します。
7. 生成された `キーワード`、`学び領域`、`授業` シートの文章を必要に応じて調整します。
8. `デプロイ > 新しいデプロイ > ウェブアプリ` から公開URLを発行します。

`preview.html` は画面確認用です。本番では `Code.gs` と `Index.html` をApps Scriptに入れて使います。

## 当日の使い方

タブレットやPCでWebアプリURLを開き、来場者に気になるキーワードを選んでもらいます。結果画面は会話のきっかけとして使い、詳しい履修や時間割は必要になった時だけ補足します。

## 編集のコツ

- キーワードを増やす場合は、`キーワード` シートに行を追加します。
- `対応領域` は `M:3,DS:1` のように、領域コードと点数をカンマ区切りで書きます。
- 領域コードは `G`, `MD`, `M`, `FA`, `PD`, `P`, `SA`, `DS` を使います。
- 結果文を変える場合は、`学び領域` シートの説明、問い、相談で聞くことを編集します。
