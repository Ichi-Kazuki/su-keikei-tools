# 経済経営学部 オープンキャンパス診断Webアプリ

高校生・保護者向けの「未来をデザインする3分診断」です。6問の回答から、経済経営学部での学びを6つのテーマで提案します。個人情報の入力・保存・メール送信は行いません。

## できること

- 登録不要で診断結果を表示
- 「国際・グローバル」「ビジネスデザイン」「金融・財務」「政策・地域・SDGs」「データサイエンス」「リーダーシップ」の6領域から学びを提案
- 学部説明や在学生相談で、診断結果をもとに質問するためのヒントを表示
- スマートフォン対応、診断結果の印刷・保存に対応

## デプロイ手順

1. [Google Apps Script](https://script.google.com/) で新しいプロジェクトを作成します。
2. プロジェクトのタイムゾーンを `Asia/Tokyo` に設定します。
3. 次の5ファイルを同名で作成し、このフォルダ内の内容を貼り付けます。
   - `Code.gs`
   - `Index.html`
   - `Styles.html`
   - `JavaScript.html`
   - `appsscript.json`
4. **デプロイ** → **新しいデプロイ** → 種類で **ウェブアプリ** を選びます。
5. 利用者がアクセスできる範囲を設定し、初回の認可を完了します。
6. 発行されたURLを、オープンキャンパス案内やQRコードへ設定します。

## 運用前チェック

1. 診断を最後まで進め、結果の内容・スマートフォン表示を確認する。
2. 左上の学部ロゴを押し、どの画面からでも診断のトップ画面へ戻れることを確認する。
3. 結果の印刷・保存が必要な運用かを確認する。
4. 開催日・会場・相談ブースなどを画面へ追加する場合は、確定した情報だけを掲載する。

## 調整しやすい場所

- 診断の質問・配点・結果文：`JavaScript.html`
- 色・レイアウト：`Styles.html`
- Webアプリの入口：`Code.gs`
